README
======

It is highly recommended to read this file before starting the
development of your project. It is also highly recommended to study
the inline comments in the provided source files.

Put your code in Main.java file and implement the required methods.
You can put the classes which describe the given structures as
sub-classes in the Main class. This is suggested to avoid the usage of
packages, but you are free to make as you want.

You have to know how to compile and run the project.

Compilation:
------------

    javac Main.java

Execution:
----------

    java Main <input_file>
